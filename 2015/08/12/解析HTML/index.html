#解析HTML

前言


这篇文章是由本人从教课书以及网站w3school中总结而来（w3school是一个学习HTML的极佳的网站，大家可以去那里学习相关知识），尽可能地用自己通俗的语言去表达各个概念的意思以方便初学者的理解，本文包含了HTML中近乎所有的知识点，如有错误或者不足之处，欢迎在留言中指出。

HTML介绍


在这个全民联网的时代，HTML大家一定非常熟悉了，当然，这里的熟悉也只是建立在眼熟的程度，大多数跟HTML的关系都只是见面打个招呼的关系，这当然不行，所以我就来帮助大家深入来了解HTML，帮助大家探寻HTML的秘密。

超文本标记语言


如果你要去百度HTML这个词语的话，你会发现它是超文本标记语言,不要被这个看起来很可怕的词语给吓到，我们一半一半地进行分析，首先是超文本，超文本是什么意思呢，百科上这样说：超文本是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本。就在这篇文章之中就已经存在了很多的超文本，比如说目录上面的字都是超级文本，它们可以实现跳转功能，比如说我在前言中的w3school也是一个超文本，可以跳转到该网站的主页，这样一解释的话，可能大家就可以理解什么事超文本了。接下来我们来解释标记语言，标记语言，是一种将文本以及文本相关的其他信息结合起来，展现出关于文档结构和数据处理细节的电脑文字编码。
Q:标记语言是什么，同我之前学习的c语言一样吗？
A：标记语言知识一种描述文本信息的语言，c语言不是标记语言，从某种意义上来说，HTML就是网页，它的代码本身仅仅是在描述网页的布局。
这样一来，我们便可以理解什么是超文本标记语言了，这便是一个很好的基础。

HTML的发展历史


实际上，发展历史什么的并没有什么用，在这里我就不和教科书一样吧啦吧啦那些东西了，不过有些重要的需要科普一下，那就是HTML和*XHTML之间的关系。HTML本身的语法要求非常低，甚至出错了依然可以正常运行，程序仅仅是忽略那出错的一行。XHTML是HTML语法的完善版，其严格规定了语法，但是目前网络上的网站如果要按照其语法来的话，大部分都是出错的，我们学习的时候按照XHTML的语法学习会比较好，尽管有时候你不那样去做也不会出错，但是难保在未来的更新中会出现错误。

如何书写和运行HTML程序


实际上，现在存在很多HTML的编译器，不过就学习而言，没有什么会比单纯的文本编辑应用要好，推荐使用notepad++，当然你也可以去用你喜欢的。记事本也可以用。在里面写好你的程序之后，格式转换为HTML，然后用浏览器便可以运行了。

3.HTML文档的标准结构


在这里我们将了解一个完整的HTML文档所应该具有的结构。首先我们先来看一下完整的结构，然后我们一点一点来解析。

<!DOCTYPE html>
<html>
<head>
<title>我的标题</titile>
</head>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html>

第一行的作用是声明这个文档是HTML文档。后面就是文档的内容 了。一个HTML文档包含两部分，一个是head部分，一个是主体部分，也就是body部分。head部分和主体部分可以包含其所对应的元素。这就是HTML文档的结构，我们的HTML文档也是按照这种结构来书写的，上面的标签和元素如果不理解也没有关系，可以在后面慢慢学到。

HTML标签和基本元素及其属性


HTML中所有标签列表

标签


标签通常由开始标签<标签类型>,结束标签 < /标签类型>组成，在开始标签和结束标签中可以放置标签的内容。
当然也有特殊标签的存在，这种标签内没有内容，表示特定的意思。例如< br/>标签的作用是换行，类似于我们编程语言中的’\n’，< hr>标签的作用是绘制一条水平的分割线。

基本元素及其属性


元素指的是从开始标签到结束标签的所有代码，可以在开始标签中指定该元素所具有的属性，格式为属性类型=”属性值“，属性特别多，我们在这里不一一讨论，点我查看完整的属性列表。

元素


p元素是html标签中一个比较常见的元素，作用是标示出一个段落，p标签中间的内容就是段落的内容。我来举个简单的例子：

<p> 这是一个简单的段落 </p>

显示效果如下：

这是一个简单的段落

这一行简单的代码便会输出一个段落出来。

h元素


h元素起的是标题作用，用来声明标题，HTML中有6个标题，分别为h1,h2,h3,h4,h5,h6，其中h1是最大的一个标题标签，h6是最小的标题标签。可以看一下下面的简单例子（本例子来源于w3school）：

<html>
<body>
<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>
<p>请仅仅把标题标签用于标题文本。不要仅仅为了产生粗体文本而使用它们。请使用其它标签或 CSS 代替。</p>
</body>
</html>

最后会显示6个大小不同的标题标签。

结构元素


这里包含了我们之前在结构中所看到的一些元素，有<html>，<body>,<head>元素,html元素定义了整个HTML文档，head元素中定义了文档标题，样式表信息等重要内容，body是我们的文档主题，我们的代码大部分都是在body中完成的。

链接元素


通过链接元素可以指定一段文字进行连接跳转，比如说跳转到百度主页，我们是通过a元素来实现跳转的。

文字链接


我们就拿跳转到百度主页来举例子：

<a href="http://www.baidu.com">跳转到百度主页</a>

在a标签中我们指定了标签的一个必要的属性，那就是需要跳转的页面的url，标签的内容是跳转到百度主页这一段文字。

4.2.7.2.图片链接


当然，我们也可以使用图片进行超链接，我们只需要把跳转到百度主页这几个字替换成一个图像元素就可以了。

<img  src="http://i3.tietuku.com/d2ffc34877f56c50.jpg" />

比如这一行代码：

<h4 id="tiaozhuan">4.2.8.图像元素</h4>

图像元素


图像元素由标签和属性指定。img标签比较特殊，是一个空标签，没有开始标签和结束标签，用<img/>表示，通常会加上src属性，所以基本格式为<img src="url" />,图像元素通常还会有alt属性，来表示如果图像加载失败之后会显示的文字信息。将鼠标放在图片的上面也会显示文字。
<img src="url" alt="提示文字信息">

自定义类型的列表


除了单纯的有序无序。html中还可以自己去定义列表的类型，该类型由<dl>创建，<dt>代表自定义的列表序号，<dd>代表列表的内容，其实我们可以想象我们的单词表便是一个定义列表，单词是dt，单词解释是dd，举个例子：

<dl>
<dt>Coffee</dt>
<dd>Black hot drink</dd>
<dt>Milk</dt>

</dl>
