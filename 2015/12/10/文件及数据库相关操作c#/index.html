c#文件及数据库相关操作

  • c#

这里面涉及到的知识点有c#文件的读写，文件的异或加密，sqlserver数据库的读入以及获取数据到Listbox和DataGridView以及TextBox.

c#文件操作

读文件

通过StreamReader来实现

StreamReader sr = new StreamReader("路径",Encoding.Default);
String line;
while((line=sr.ReadLine())!=null)
{
    Console.WriteLine(line.ToString());
}
sr.Close();

写文件

通过StreamWriter来实现

StreamWriter构造函数的第二个参数是是否追加，true表示追加写入文件，false表示不追加写入文件。

StreamWriter sw = new StreamWriter("路径",false,Encoding.Default);
sw.Write("写的内容"); 
sw.WriteLine("写的内容");
sw.Close();

write和writeline的区别在于有没有换行的存在.

文件异或加密与解密

原理是某个数异或同一个数两次之后，得到的数还是这个数本身。

string s = "i wanna see you!";
char[] ss = s.ToCharArray();//将s字符串转化为字符数组
char a = '#';//加密使用的字符，即秘钥
//加密
for (int i = 0; i < ss.Length; i++)
    ss[i] = (char)(ss[i] ^ a);//将每一位都根据秘钥进行了异或加密
for (int i = 0; i < ss.Length; i++)
    Console.Write(ss[i] + "");//输出加密结果
Console.WriteLine();
//解密
for (int i = 0; i < ss.Length; i++)
    ss[i] = (char)(ss[i] ^ a);//将每一位都根据秘钥进行了异或解密
for (int i = 0; i < ss.Length; i++)
    Console.Write(ss[i] + "");
Console.Read();

例子

从A.txt文本文件中按照行读取字符串，进行加密，把加密后的字符串写到B.txt文件中，然后把A.txt文件中的内容删除，读出B.txt中的加密字符串，并解密，把解密的字符串回写到A.txt中

StreamReader sr = new StreamReader("A.txt",Encoding.Default);//放在debug目录下不需要写其余路径
string s=sr.ReadToEnd();
sr.Close();    
//加密
char a = '#';
char[] ss = s.ToCharArray();
for (int i = 0; i < ss.Length; i++)
    ss[i] = (char)(ss[i] ^ a);
//写入B.TXT
StreamWriter sw = new StreamWriter("B.txt", false, Encoding.Default);
sw.Write(ss);
sw.Close();
//删除A.txt中的内容
StreamWriter sw1 = new StreamWriter("A.txt", false, Encoding.Default);
sw1.Write("");//写入空字符串,相当于删除了原文件内容
//因为后面还要写入，这里我们就不关闭了
//读出B.txt的内容
StreamReader sr1 = new StreamReader("B.txt", Encoding.Default);
string s1 = sr1.ReadToEnd();
sr1.Close();
//解密
char[] ss1 = s1.ToCharArray();
for (int i = 0; i < ss1.Length; i++)
    ss1[i] = (char)(ss1[i] ^ a);
//写入A.txt
sw1.Write(ss1);
sw1.Close();

c#连接数据库操作

  1. 创建新的数据库连接SqlConnection，并打开数据库
  2. 创建对数据库的操作SqlCommand,连接数据库，写好查询语句

  3. 创建DataSet和数据库之间的桥梁SqlDataAdapter并连接好数据库以及DataSet

    //第一步
    SqlConnection conn = new SqlConnection(“连接字符串”);
    conn.open();
    //第二步
    SqlCommand cmd = new SqlCommand();
    cmd.Connection = conn;
    cmd.CommandText = “Select * from student”;
    //第三步
    SqlDataAdapter DataAdapter = null;
    DataSet dataset = null;
    DataAdapter.SelectCommand = cmd;
    DataAdapter.Fill(dataset, “t1”);//t1为dataset中的表名

绑定数据到listbox

//获取所有列名
string s="";
for (int i = 0; i < dataset.Tables["t1"].Columns.Count; i++)
    s += dataset.Tables["t1"].Columns[i].ToString() + "\t";
listBox1.Items.Add(s);
//获取剩余数据项
for (int i = 0; i < dataset.Tables[0].Rows.Count; i++)
{
    DataRow dr = dataset.Tables[0].Rows[i];
    s = "";
    for (int j = 0; j < dataset.Tables[0].Columns.Count; j++)
        s += dr[j].ToString() + "\t";
    listBox1.Items.Add(s);
}

绑定数据到textBox

//将t1表中的t_name属性绑定到textBox1
textBox1.DataBindings.Add("Text",dataset,"t1.t_name");

绑定数据到DataGridView

dataGridView1.DataSource = dataset;
dataGridView1.DataMember = "t1";
